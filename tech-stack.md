
# Technical Stack

## Frontend

### Core
- **React**: v19 - JavaScript library for building user interfaces.
- **Vite**: v7 - Build tool and development server.
- **JavaScript (ES Modules)**: Language.

### Styling & UI
- **TailwindCSS**: v4 - Utility-first CSS framework.
- **PostCSS**: Tool for transforming CSS with JavaScript.
- **Autoprefixer**: PostCSS plugin to parse CSS and add vendor prefixes.
- **Lucide React**: Icon library.
- **@tabler/icons-react**: Icon library.
- **clsx**: Utility for constructing `className` strings conditionally.
- **tailwind-merge**: Utility to merge Tailwind CSS classes without conflicts.

### Animation & 3D
- **Motion**: Animation library (formerly Framer Motion).
- **Three.js**: 3D library.
- **@react-three/fiber**: React renderer for Three.js.
- **TSParticles**: Particle system for backgrounds and effects.
- **postprocessing**: Post-processing effects for Three.js.

### Components & Utilities
- **React Router DOM**: v7 - Declarative routing.
- **React Markdown**: Markdown renderer.
- **React Player**: Video player component.
- **React Quill**: Rich text editor.
- **Excalidraw**: Virtual whiteboard component.

---

## Backend

### Core Framework
- **Flask**: Micro web framework for Python.
- **Flask-CORS**: Extension for handling Cross Origin Resource Sharing (CORS).

### Database
- **MongoDB (PyMongo)**: NoSQL database driver.
- **ChromaDB**: AI-native open-source embedding database (Vector DB).

### AI & Machine Learning
- **OpenAI**: Client library for accessing OpenAI API.
- **Google API Python Client**: Client library for Google's discovery based APIs.

### Authentication & Security
- **PyJWT**: JSON Web Token implementation.
- **Bcrypt**: Library for hashing passwords.

### Data Processing & ingestion
- **Pandas**: Data analysis and manipulation tool.
- **NumPy**: Fundamental package for scientific computing.
- **PyPDF**: PDF toolkit for extracting text and more.
- **Feedparser**: Universal feed parser (RSS/Atom).

### Utilities & Services
- **Cloudinary**: Cloud service for image/video management.
- **Edge-TTS**: Text-to-speech library.
- **Python-Dotenv**: Reads key-value pairs from `.env` file.
- **Requests**: HTTP library for Python.
